                        🏛️ AIDCONNECT - PROJECT OVERVIEW 🏛️
                     Grant-in-Aid Application Portal System
===============================================================================

📋 PROJECT DESCRIPTION
-----------------------
AidConnect is a comprehensive government portal system designed for transparent 
and efficient beneficiary identification and grant-in-aid application processing.
The system facilitates seamless interaction between citizens, nodal officers, 
and administrators for scheme management and application processing.

🌟 ANIMATION TECHNIQUES USED
----------------------------
1. CSS TRANSITIONS & TRANSFORMS
   • hover:scale-105 - Smooth scaling effects on buttons and cards
   • transition-all duration-300 - Smooth property transitions
   • hover:shadow-lg - Dynamic shadow effects

2. INTERSECTION OBSERVER API
   • Scroll-triggered animations for sections
   • Fade-in effects when elements enter viewport
   • Staggered animations with custom delays

3. CUSTOM CSS ANIMATIONS
   • animate-fadeInUp - Bottom-to-top fade animations
   • animate-fadeInRight - Right-to-left slide animations
   • animate-slideDown - Top-to-bottom slide effects
   • pulse-animation - Breathing effect for interactive elements

4. ANIMATION IMPLEMENTATION
   ```javascript
   useEffect(() => {
     const observer = new IntersectionObserver(
       (entries) => {
         entries.forEach((entry) => {
           if (entry.isIntersecting) {
             setIsVisible(prev => ({ ...prev, [entry.target.id]: true }));
           }
         });
       },
       { threshold: 0.1 }
     );
   }, []);
   ```

5. TAILWIND ANIMATION CLASSES
   • Custom animation delays: animation-delay-200, animation-delay-400
   • Transform utilities: hover:scale-110, hover:rotate-3
   • Transition utilities: transition-all, duration-200, ease-in-out

===============================================================================
                           🗄️ DATABASE DESIGN
===============================================================================

📊 MONGODB COLLECTIONS
----------------------
1. USERS COLLECTION
   • Personal Information (name, email, mobile, address)
   • Authentication Data (password hash, verification status)
   • Document References (photo, signature, certificates)
   • Timestamps (createdAt, updatedAt, firstSignedUp)

2. APPLICATIONS COLLECTION
   • User Reference (userId)
   • Scheme Information (schemeTitle, schemeId)
   • Application Status (pending, approved, rejected)
   • Document Attachments (aadhaar, panCard, residential certificate)
   • Processing Data (applicationDate, remarks, approvedBy)

3. SCHEMES COLLECTION
   • Scheme Details (title, description, category)
   • Timeline (startDate, endDate, correctionWindow)
   • Metadata (priority, status, feeDetails)
   • Administrative Info (createdBy, updatedAt)

🔗 DATABASE CONNECTION
----------------------
```javascript
mongoose.connect(process.env.MONGODB_URI)
  .then(() => console.log('✅ MongoDB connected'))
  .catch(err => console.log('❌ MongoDB error:', err.message));
```
